此模组用于移除模组包 "危险世界求生" 中的反作弊系统。

使用时应感觉像无缝体验，因为我能找到的所有反作弊检测都已被移除。

另外，我们新增了一个命令 `/overwrite_enable_command enable me`，可用于在创建世界时因反作弊系统导致未启用命令的单人世界中临时启用作弊。这并不是一个很好或优雅的补丁。我仍然建议在世界创建后使用 NBT 编辑器来启用单人世界的作弊。这个方法可行但不推荐，主要因为不够优雅。由于关卡数据是只读的，我无法在不大量混入代码的情况下修改它。你也可以下载一些允许直接完成此操作的模组。

## 安装步骤
1. 从发布页面下载模组 jar 文件。
2. 将 jar 文件放入 "mods" 文件夹中。
3. 启动游戏。
